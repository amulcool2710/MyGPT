## 🚀 Features & Achievements
- **Conversational AI:** Integrated external APIs to provide intelligent responses with **99% uptime**.
- **Responsive UI:** Implemented a highly responsive user interface, improving user retention and session duration by **25%**.
- **Secure Sessions:** Engineered secure session management and data handling, successfully protecting **1000+ user interactions** and enhancing reliability.

## 🛠 Setup Instructions
1. Clone the repository.
2. In the `Backend` directory, configure your `.env` variables (e.g., `OPENAI_API_KEY`). Run `npm install` and `npm start`.
3. In the `Frontend` directory, run `npm install` and `npm start`.
